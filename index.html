Commands to run after creating new git hub repo

  1. git init
  2. git add .
  3. git commit -m "message"
  4. git branch -M main
  5. git remote add origin https://github.com/userName/repoName.git
  6. git push -u origin main

Command to run everytime after making any changes

  1. git add .
  2. git commit -m "message"
  3. git push
